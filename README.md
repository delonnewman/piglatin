NAME
====
piglatin - A toy CLI piglatin translator

SYNOPSIS
========

	> piglatin
	this is some text
	C^D
	isthay isay omesay exttay

	> cat sometext.txt | piglatin

	
DESCRIPTION
===========
Takes text from standard input and translates it into Pig Latin.
(see http://en.wikipedia.org/wiki/Pig_Latin)
